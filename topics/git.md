## よく使うコマンド

```bash
# 初期化
git init

# ステージング
git add .

# コミット
git commit -m "message"

# プッシュ
git push origin main

# プル
git pull origin main

# ブランチ作成・切り替え
git checkout -b feature-branch

# マージ
git merge feature-branch
```
